 Expense Tracker

A simple and user-friendly web application to track your income and expenses.



 Features

 Add income and expense transactions
 Delete transactions
 Automatic balance calculation
 Data saved using browser localStorage (does not disappear on refresh)



 Tech Stack

* HTML
* CSS
* JavaScript



How to Run

1. Download or clone this repository
2. Open the project folder in Visual Studio Code
3. Run `index.html` using Live Server
   OR
4. Simply open `index.html` in your browser
